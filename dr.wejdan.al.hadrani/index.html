كيفية إعداد الاتصال بالسحابة

  1. اذهب إلى JSONBin.io وأنشئ حساب مجاني
  2. أنشئ Bin جديد وانسخ الـ Bin ID
  3. اذهب إلى API Keys وانسخ Master Key
  4. استبدل القيم في الكود أدناه بمفاتيحك الفعلية:
const JSONBIN_CONFIG = {
    BIN_ID: 'ضع_Bin_ID_الفعال_هنا',
    API_KEY: 'ضع_Master_Key_الفعال_هنا'
};

سيعمل النظام محلياً بدون هذه المفاتيح، لكن للمزامنة العالمية تحتاج إلى إعدادها.
